# 百度AI语音对话配置说明

## 概述
本项目已集成百度AI的语音识别、语音合成和文心一言大模型服务，实现完整的语音对话功能。

## 配置步骤

### 1. 获取百度AI API密钥

1. 访问 [百度AI开放平台](https://ai.baidu.com/)
2. 注册/登录账号
3. 创建应用，获取以下信息：
   - APP_ID
   - API_KEY
   - SECRET_KEY

### 2. 配置后端环境变量

编辑 `server/.env` 文件，填入真实的百度AI密钥：

```env
# 百度AI配置
BAIDU_API_KEY=你的百度API_KEY
BAIDU_SECRET_KEY=你的百度SECRET_KEY
```

### 3. 配置前端API密钥

编辑 `client/src/utils/baidu-ai.js` 文件，更新配置：

```javascript
const BAIDU_CONFIG = {
  APP_ID: '你的APP_ID',
  API_KEY: '你的API_KEY',
  SECRET_KEY: '你的SECRET_KEY'
};
```

## 功能说明

### 语音识别
- 支持中文语音识别
- 采样率：16kHz
- 格式：WAV

### 语音合成
- 支持中文语音合成
- 可调节语速、音调、音量
- 默认使用女声发音

### AI对话
- 使用百度文心一言大模型
- 专门优化为老年人陪伴助手"暖阳"
- 温和、耐心、易懂的对话风格

## 测试步骤

1. 确保后端和前端都已启动
2. 在浏览器中访问前端应用
3. 进入AI陪聊页面
4. 按住语音按钮开始录音
5. 松开按钮自动进行语音识别和AI回复
6. AI回复会自动进行语音合成播放

## 注意事项

- 语音功能需要用户开启麦克风权限
- 首次使用可能需要等待语音模型加载
- 如果API调用失败，会自动降级到文字回复
- 建议在网络稳定的环境下使用

## 故障排除

1. **语音识别失败**：检查麦克风权限和网络连接
2. **语音合成失败**：检查API密钥配置是否正确
3. **AI回复失败**：检查百度AI服务状态和配额

## 安全提醒

- 请妥善保管API密钥，不要在代码中明文存储
- 建议在生产环境中使用环境变量管理密钥
- 定期检查API使用量，避免超出免费额度
